# Graph Algorithms

This repository contains implementations of essential **graph algorithms** written in Python.
This project is part of the **Graph Algorithms** course at the **National Polytechnic University of Armenia(NPUA)**.

---

| File                      | Description                                                                   |
|---------------------------|-------------------------------------------------------------------------------|
| [BFS.py](https://github.com/MariHovhannisyan/Algorithms_and_DataStructures/blob/master/BFS.py)                | Breadth-First Search (BFS) algorithm for traversing graphs.                   |
| [DFS.py](https://github.com/MariHovhannisyan/Algorithms_and_DataStructures/blob/master/DFS.py)                  | Depth-First Search (DFS) traversal implementation.                            |
| [Dijkstra's.py](https://github.com/MariHovhannisyan/Algorithms_and_DataStructures/blob/master/Dijkstra's.py)           | Dijkstra's algorithm for shortest paths using a lazy priority queue approach. |
| [BellmanFord.py](https://github.com/MariHovhannisyan/Algorithms_and_DataStructures/blob/master/BellmanFord.py)          | Bellman-Ford algorithm for shortest paths, with negative cycle detection.     |
| [FloydWarshall.py](https://github.com/MariHovhannisyan/Algorithms_and_DataStructures/blob/master/FloydWarshall.py)        | Floyd-Warshall algorithm for all-pairs shortest paths.                        |
| [Kahn's.py](https://github.com/MariHovhannisyan/Algorithms_and_DataStructures/blob/master/Kahn's.py)               | Kahn's algorithm for topological sorting using in-degrees.                    |
| [TopSort.py](https://github.com/MariHovhannisyan/Algorithms_and_DataStructures/blob/master/TopSort.py)              | DFS-based Topological Sorting implementation.                                 |
| [Longest_path.py](https://github.com/MariHovhannisyan/Algorithms_and_DataStructures/blob/master/Longest_path.py)         | Finds the longest path in a Directed Acyclic Graph (DAG).                     |
| [connected_components.py](https://github.com/MariHovhannisyan/Algorithms_and_DataStructures/blob/master/connected_components.py) | Identifies connected components in an undirected graph.                       |
| [Dungeon.py](https://github.com/MariHovhannisyan/Algorithms_and_DataStructures/blob/master/Dungeon.py)              | Finds the minimum steps to navigate a maze from start ('S') to end ('E').     |
